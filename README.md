# dotfiles

Personal config files for an [Omarchy](https://omarchy.org/) system (Arch Linux + Hyprland).

## Contents

| Path | Description |
|------|-------------|
| `hypr/` | Hyprland window manager config (bindings, monitors, appearance, idle, lock screen) |
| `waybar/` | Status bar layout and styles |
| `alacritty/` | Alacritty terminal |
| `kitty/` | Kitty terminal |
| `ghostty/` | Ghostty terminal |
| `mako/` | Notification daemon |
| `walker/` | App launcher |
| `starship.toml` | Shell prompt |
| `omarchy/branding/` | Omarchy branding customizations |

## Notes

- Hypr shaders live in a separate repo: [hyprland-glitch-shader](../hyprland-glitch-shader/)
- All files map to `~/.config/<path>` on the target system
